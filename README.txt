CobraGER Red/Black English Website

How to use:
1. Unzip the file.
2. Open index.html in your browser.
3. Replace the Discord button link in index.html with your own invite.
4. Edit names, roles, game sections, and social links.
5. Upload the full folder to your web host.

Files:
- index.html: website content
- styles.css: red/black design and layout
- script.js: mobile menu
- assets/logos/: logo image
